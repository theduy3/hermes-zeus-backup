# Media Retrieval and Content

## Overview

This umbrella covers fetching online media-derived text/assets and turning them into usable content. Detailed YouTube and GIF workflows are preserved under `references/source-packages/`.

## When to Use

- Fetch YouTube transcripts and summarize, thread, or blog them.
- Search or download GIFs from Tenor.
- Produce content from retrieved media evidence.

## Verification Checklist

- [ ] Source URL/query captured.
- [ ] Retrieval succeeded with real transcript/asset metadata.
- [ ] Output format selected: summary, thread, blog, file path, or metadata.
