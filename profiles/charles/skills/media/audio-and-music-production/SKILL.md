# Audio and Music Production

## Overview

This umbrella covers music/audio creation and analysis. It separates creative songwriting decisions from model/tool execution while preserving detailed source packages under `references/source-packages/`.

## When to Use

- Write lyrics, structure a song, or craft AI-music prompts.
- Generate music or sound effects with AudioCraft/MusicGen or HeartMuLa.
- Analyze audio with spectrograms, mel/chroma/MFCC features, or related visualizations.

## Subsections

### Songwriting and AI music prompts
Use the songwriting reference for structure, rhyme/meter, emotional arc, Suno tags, and phonetic tricks.

### HeartMuLa
Use for HeartMuLa installation and Suno-like generation workflows.

### AudioCraft / MusicGen / AudioGen
Use for local model-based text-to-music/text-to-sound generation, model selection, GPU requirements, and troubleshooting.

### SongSee
Use for audio spectrogram and feature extraction workflows.

- [ ] Output format specified: lyrics, prompt, WAV/MP3, visualization, or feature data.
- [ ] Generation/analysis tool availability checked when executing.
- [ ] Audio artifacts saved and paths returned when generated.
