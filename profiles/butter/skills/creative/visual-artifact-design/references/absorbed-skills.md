# Absorbed skill packages

Consolidated visual artifact skills:

- `architecture-diagram` — dark-themed SVG/HTML architecture diagrams and template.
- `claude-design` — one-off HTML artifacts such as landing pages, decks, and prototypes.
- `design-md` — DESIGN.md token spec authoring/validation/export.
- `excalidraw` — hand-drawn Excalidraw JSON diagrams and upload/colors examples.
- `popular-web-designs` — real-world HTML/CSS design-system templates.
- `sketch` — quick throwaway HTML mockups with variant comparison.
- `p5js` — p5.js generative/interactive/WebGL sketches and render/export scripts.
- `pretext` — DOM-free text layout, kinetic typography, and text-as-geometry demos.
- `baoyu-infographic` — infographic prompt framework with layouts/styles.

Large support packages were archived intact for recoverability rather than flattened into broken references.
