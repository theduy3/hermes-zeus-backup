# Absorbed skill packages

The following former standalone skills were consolidated into this umbrella and archived as complete packages for recoverability:

- `github-auth` — authentication, SSH/HTTPS, gh login, credential helper setup.
- `github-repo-management` — clone/create/fork repositories, remotes, releases, API cheatsheet.
- `github-issues` — issue creation, triage, labeling, templates.
- `github-pr-workflow` — branch/commit/open PR/check CI/merge lifecycle and PR body templates.
- `github-code-review` — review PR diffs and posting inline comments, including output template.

Support files remain preserved inside each archived source package; this umbrella captures the class-level workflow for discovery.
