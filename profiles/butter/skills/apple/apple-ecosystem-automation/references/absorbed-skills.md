# Absorbed skill packages

- `apple-notes` — Apple Notes via memo CLI.
- `apple-reminders` — Reminders via remindctl.
- `imessage` — iMessage/SMS via imsg.
- `findmy` — Find My device/AirTag location on macOS.
- `macos-computer-use` — background desktop screenshots and input control.

The umbrella groups Apple/macOS automation by class while archived packages preserve command-level specifics.
