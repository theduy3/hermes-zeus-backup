# Absorbed Agent CLI Skills

This reference records external coding agent CLI skills consolidated into `coding-agent-delegation`.

- `claude-code` — Claude Code CLI print mode, PTY/tmux interactive mode, workspace trust dialogs, and PR-oriented delegation.
- `codex` — OpenAI Codex CLI one-shot tasks, background mode for long tasks, key flags, and PR review use cases.
- `opencode` — OpenCode CLI binary resolution, one-shot implementation/review, and interactive session handling.

The umbrella keeps the shared supervision and verification rules; archived packages preserve exact CLI command details.
