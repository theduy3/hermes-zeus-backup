# Absorbed GitHub Skills

This reference records narrow GitHub skills consolidated into `github-workflows`.

- `github-auth` — detection flow for git-only HTTPS/PAT, SSH, and `gh` CLI authentication; included `scripts/gh-env.sh` in the archived package.
- `github-repo-management` — clone/create/fork/template/remotes/releases; archived package includes a GitHub API cheatsheet.
- `github-issues` — issue viewing/creation/triage and bug/feature templates.
- `github-pr-workflow` — branch/commit/push/PR/CI/merge lifecycle, PR body templates, CI troubleshooting, conventional commits.
- `github-code-review` — local and PR diff review, setup for PR interactions, review output template.
- `codebase-inspection` — LOC/language/codebase composition using tools such as pygount/cloc/tokei with exclusions.

Archived packages retain original templates/scripts/references unchanged.
