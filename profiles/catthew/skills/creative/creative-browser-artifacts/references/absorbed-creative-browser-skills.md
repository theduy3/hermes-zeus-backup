# Absorbed Creative Browser Artifact Skills

This reference records browser/diagram/design skills consolidated into `creative-browser-artifacts`.

- `architecture-diagram` — dark themed SVG/HTML architecture/cloud/infra diagrams with semantic color and visual language.
- `claude-design` — one-off HTML artifacts such as landing pages, decks, and prototypes.
- `sketch` — throwaway HTML mockups with 2–3 variants for comparison.
- `popular-web-designs` — design-system inspiration catalog and HTML generation patterns.
- `pretext` — creative browser demos using text layout and text-as-geometry patterns.
- `p5js` — p5.js generative sketches, interactive/3D demos, and export pipeline; archived package keeps templates/scripts.
- `excalidraw` — hand-drawn Excalidraw JSON diagrams plus upload script and examples.
- `design-md` — Google's DESIGN.md token spec authoring, validation, and starter template.

Archived packages retain any templates/scripts/references that were too detailed for the umbrella.
