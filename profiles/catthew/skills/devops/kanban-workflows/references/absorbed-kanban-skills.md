# Absorbed Kanban Skills

This reference records Kanban role skills consolidated into `kanban-workflows`.

- `kanban-orchestrator` — decomposition playbook, anti-temptation rules, task graph sketching, and routing work through the board.
- `kanban-worker` — workspace handling, tenant isolation, summaries/metadata, claim discipline, block reasons, and heartbeat guidance.

The umbrella combines both roles while preserving role-specific subsections.
