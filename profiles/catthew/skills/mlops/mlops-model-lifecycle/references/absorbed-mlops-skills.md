# Absorbed MLOps Skills

This reference records MLOps skills consolidated into `mlops-model-lifecycle`.

- `huggingface-hub` — `hf` CLI authentication, model/dataset search, download, upload, and repo management.
- `llama-cpp` — local GGUF inference, HF Hub discovery, server mode, optimization, troubleshooting, and quantization references.
- `vllm` / candidate `serving-llms-vllm` — OpenAI-compatible serving, offline batch inference, quantization, optimization, and deployment troubleshooting.
- `lm-evaluation-harness` / candidate `evaluating-llms-harness` — standard and custom LLM benchmark evaluation, distributed/API evaluation, and benchmark guidance.
- `weights-and-biases` — experiment tracking, sweeps, artifacts, integrations, and model registry workflows.
- `segment-anything` / candidate `segment-anything-model` — SAM checkpoints and zero-shot segmentation operational setup.

Archived packages retain detailed references/templates for exact commands.
