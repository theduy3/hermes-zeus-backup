# Absorbed Apple Skills

This reference records the app-specific skills consolidated into `apple-ecosystem`.

- `apple-notes` — `memo` CLI workflows for creating, searching, viewing, and editing Apple Notes.
- `apple-reminders` — `remindctl` workflows for listing, adding, completing, deleting, and managing reminder lists.
- `imessage` — `imsg` workflows for listing chats, reading history, and sending iMessage/SMS while avoiding recipient mistakes.
- `findmy` — Find My workflows using AppleScript, screenshots, and UI automation to inspect device/AirTag location state.
- `macos-computer-use` — canonical screenshot/action loop for operating macOS apps with background rules, text input patterns, and drag/drop handling.

Archived packages remain recoverable if exact command tables are needed later.
